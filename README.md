# 組織図管理システム - 株式会社ソウルシンクス

## 概要

社員・部署管理、組織図表示、フィルター機能を備えた組織図管理システムです。

## 機能

- 社員・部署の追加・編集・削除
- 組織図の表示（カード/ツリー/リスト）
- フィルター機能（部署、雇用形態、スキル、在籍状況）
- エクスポート/インポート機能
- PDF出力
- 統計表示
- Chatwork通知連携

## 技術スタック

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- Font Awesome

## デプロイ

単一HTMLファイルで動作します。Webサーバーに`index.html`をアップロードしてください。
